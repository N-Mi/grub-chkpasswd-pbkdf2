# grub-chkpasswd-pbkdf2
Small python tool to check an encrypted GRUB2 password generated by grub-mkpasswd-pbkdf2


## Usage

Given a hash generated by `grub-chkpasswd-pbkdf2` :
```
# password in this example is "test123"
$ grub-mkpasswd-pbkdf2 
Enter password: 
Reenter password: 
PBKDF2 hash of your password is grub.pbkdf2.sha512.10000.A8B569CF614401BDC3A6FE20E8088D64C5042AFFA4761BEE1108955A73F517D99CDC58AD353C6A5D175C8CC0D6255338335EB97D85B5527BB5ECA08435DA9E86.6F45F2E8B81139E96AF9D19B5D4C0FA92B6AE4F213A844EA2353219E6C8310A21E57CABB238B2F8469FA0CCB32D9C67C155C6E77B6AFAA93661CA2DCAC8F13BC
```

Check the password associated to this hash is "test123" :

